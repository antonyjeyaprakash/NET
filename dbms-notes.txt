Assume that a read request takes 50 nsec on a cache miss and 5 nsec on a 
cache hit. While running a program, it is observed that 80% of the 
processor’s read requests result in a cache hit. The average read access 
time is ………….. nsec.
Correct answer is 14.
Average read time = 0.80 x 5 + (1 – 0.80) x 50 = 14 nsec
